1. Packages installieren 
sudo apt install gcc-erm-linux-gnueabihf libc6-dev-armhf-cross qemu-user gdb-multiarch

2. GNU Debugger Init-File

In der Datei

/etc/gdb/gdbinit

folgendes einfügen:

set auto-load safe-path /

3. Im Projektverzeichnis eine .gdbinit Datei erstellen mit Inhalt:

target remote localhost:1234
layout r

define hook-quit
	set confirm off
end

3. Shell-Skript compileAndDebugOnARM.sh <filename> ausführen

s: Stept im Code weiter (nach dem ersten s nur noch Enter nötig)
q: Schließt den Debugger
